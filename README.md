# To-Do

Este repositório contém uma aplicação simples de lista de tarefas (To-Do) feita com React.

## Funcionalidades
 - Adicionar novas tarefas.
 - Marcar tarefas como concluídas.
 - Exibir tarefas pendentes e concluídas.
 - Excluir tarefas da lista.

## Tecnologias Utilizadas
 - **React** Para a logica da aplicação.
 - **Tailwind CSS** usado para estilização.
 - **Lucide React** Biblioteca de icones.
